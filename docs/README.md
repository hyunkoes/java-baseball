# Flow

1. computer의 랜덤 넘버를 생성
2. 게임 시작
3. 플레이어의 입력
4. 결과 계산
5. 3스트라이크가 나올 때까지 3,4를 반복

# 기능 목록

### 컴퓨터
- Computer의 랜덤 넘버 생성 기능
- 게임의 흐름을 제어하는 기능 ( 종료 / 재시작 / 재입력 )
- 볼, 스트라이크를 계산하는 기능

### 사용자
- 사용자 입력
- 사용자의 입력을 검증하는 기능 ( 서로 다른 3자리 수 / 1,2 가 아닌 경우 예외 처리 ) `IllegalArgumentException`
- 게임 종료 시 재실행 여부를 물어보는 기능
